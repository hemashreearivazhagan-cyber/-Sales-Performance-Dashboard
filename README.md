# 📊 Sales Performance Dashboard | Power BI

## 📌 Project Overview
This project presents an interactive Sales Performance Dashboard built using Microsoft Power BI. The dashboard transforms raw Superstore sales data into meaningful business insights through interactive visualizations, helping users analyze sales performance, profitability, customer segments, and regional trends.

---

## 🎯 Objective
The objective of this project is to:
- Visualize business performance using interactive dashboards.
- Track key performance indicators (KPIs).
- Identify sales trends and regional performance.
- Support data-driven business decisions.

---

## 🛠️ Tools Used
- Microsoft Power BI
- Microsoft Excel
- Superstore Sales Dataset (Kaggle)

---

## 📂 Dataset
**Dataset:** Superstore Sales Dataset

The dataset contains information about:
- Orders
- Sales
- Profit
- Customers
- Regions
- Categories
- Shipping Details

---

## 📈 Dashboard Features

### ✅ KPI Cards
- Total Sales
- Total Profit
- Total Orders

### 📊 Visualizations
- Revenue Trend (Line Chart)
- Sales by Region (Bar Chart)
- Sales by Category
- Profit by Category
- Customer Segment Distribution
- Order Distribution

### 🎛️ Interactive Slicers
- Region
- Category
- Segment
- Ship Mode

---

## 📌 Key Insights
- Identified the highest-performing sales regions.
- Compared profit across different product categories.
- Observed monthly sales trends.
- Analyzed customer segmentation.
- Enabled interactive filtering for detailed analysis.
